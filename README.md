# ThreeJs-TypeScript+Webpack+glslify-TPL

## 简介：

这是一个用threejs构建webgl项目的基础框架

1 内置gulp工作流利用webpack打包

2 利用typeScript开发项目

3 内置glslify shader工具

4 利用 threeQuery框架进行threejs快速构建
